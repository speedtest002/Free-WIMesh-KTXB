# Free-WIMesh-KTXB
Script kết nối wifi nhanh chóng
Kết nối 2 tiếng thay vì 1 tiếng như mặc định, không cần tốn 5s cuộc đời để coi quảng cáo và nghe cái nhạc nhảm nhí đó
Có thể bị fix bất cứ lúc nào nên chịu thôi chứ sao :Đ
